# ArcConnect

ArcConnect is a **FastAPI + PostgreSQL** web application designed to connect students and mentors.  
It provides role‑based signup, authentication, and dashboards for both portals.

---

## 🚀 Features
- Student and Mentor signup with role validation
- Secure password hashing using bcrypt
- JWT authentication for login
- Separate dashboards for students and mentors
- Organized project structure with routers, models, schemas, and templates
